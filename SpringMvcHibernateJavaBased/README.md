## Quick Process
* 将源代码导入IDEA中
* 执行maven-archetype-templates\SpringMvcHibernateJavaBased\sql\usersdb.sql
* 部署项目，启动tomcat服务器
* web页面具体路径：http://localhost:8080/SpringMvcHibernateJavaBased
* 首页展示用户列表，可进行新增修改删除用户操作
